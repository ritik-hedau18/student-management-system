# Student Registrations Database

Student Registrations Database is a Spring Boot REST API service managing Student records.

## Features
- Enrollments management
- Course grades tracking
- Course directory links

## Running Locally
Run `mvn spring-boot:run` in the project directory. The service runs on port `8080`.
